    SELECT TOP, LIMIT, ROWNUM 子句

    介绍

    SELECT TOP 子句用于规定要返回的记录的数目。

    SELECT TOP 子句对于拥有数千条记录的大型表来说，是非常有用的。
    ** notes **:
    不是所有的数据库系统都支持 SELECT TOP 语句。
    MySQL 支持 LIMIT 语句来选取指定的条数数据， Oracle 可以使用 ROWNUM 来选取。
