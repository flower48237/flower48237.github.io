    UNION

    介绍

    用于合并两个或多个SELECT 语句的结果集。

    UNION 内部的每个 SELECT语句必须拥有相同数量的列。
    列也必须拥有相似的数据类型。
    同时，每个 SELECT 语句中的列的顺序必须相同。

    语法(UNION)

    注释：UNION 操作符默认选取不同的值。如果允许重复的值，需要使用 UNION ALL。
